# 玄枢Alpha · 价值罗盘（Value Compass）—— MVP 报告

> **进阶层功能**：基于巴菲特投资框架，对持仓基金做「持仓穿透 + 底层个股价值体检」。
>
> 用户持有的是基金/指数，而巴菲特框架分析的是**单个公司**。因此先做**持仓穿透（look-through，透视盈余法）**——把基金穿透到前十大重仓股，再对底层个股套用巴菲特框架，最后按权重汇总成「基金底层质地视图」。

- **生成时间**：2026-06-03 18:10:37
- **数据源**：AKShare（东方财富公开披露：基金季报前十大重仓 + A股财务分析指标）
- **方法论**：持仓穿透(look-through) + 巴菲特八问质地评级 + 权重汇总

## 方法论：巴菲特八问检查表

对每只穿透出的底层个股，按以下结构化检查表打分（基于 AKShare 真实财务 + 行业常识），再映射红绿灯：

| 序号 | 检查项 | 数据依据 | 权重 |
|---|---|---|---|
| Q1 | **能力圈** | 行业是否看得懂（消费=主场；高端制造=边缘；半导体/AI算力=圈外） | 一票否决（圈外直接灰） |
| Q2 | **商业模式** | 毛利率（≥40%优 / 20–40%中） | 2 分 |
| Q3 | **护城河** | ROE（≥15%护城河迹象）+ ROE 多年稳定性 | 3 分 |
| Q4 | **定价权** | 净利率（≥20%强定价权） | 2 分 |
| Q5 | **现金含金量** | 经营现金流/净利润（≥80%利润成色高） | 2 分 |
| Q6 | **抗压能力** | 资产负债率（≤50%稳健） | 1 分 |
| Q7 | **成长性** | 营收/净利增速 | 参考，非评级核心 |
| Q8 | **管理层 & 安全边际** | 内在价值估值 + 管理层诚信 | ⚠ 本 MVP 未做，已诚实标注 |

**评级映射**：
- 能力圈内 + 质地分 ≥70% → 🟢绿；45–70% → 🟡黄；<45% → 🔴红
- 能力圈边缘 → 质地再好最多 🟡黄（谨慎）
- 能力圈外 / 核心财务缺 2 项以上 → ⚪灰（不予评级）

**穿透质地汇总**：把底层个股评级按「占基金净值权重」加权，得到基金「底层质地」概述。

## 关于用户持仓中的非个股资产

用户持仓里还包括**黄金 ETF（易方达黄金 002963/159934，占比超 50%）、有色金属 ETF、货币基金（余额宝）**等。

> ⚠️ **巴菲特框架不适用于这类资产**。黄金/大宗商品不产生现金流、没有 ROE、没有护城河，巴菲特本人明确不投黄金（「黄金不会生小金子」）。货币基金是现金管理工具。这些资产应由「资产配置 / 宏观信号」维度评估（玄枢已有黄金/有色信号模块），**不纳入价值罗盘的个股质地评级**。

## 数据来源、时间与局限

- **数据源**：AKShare（聚合东方财富 / 公开季报披露），免费中国金融数据库。生成于 2026-06-03 18:10:37。
  - 基金前十大重仓：`ak.fund_portfolio_hold_em`
  - 个股财务指标：`ak.stock_financial_analysis_indicator`
  - 基金官方名称：`ak.fund_name_em`
- **数据局限（诚实标注）**：
  1. ETF 联接基金直接持股比例极低（前十大合计常 <3%，主体仓位是目标 ETF），披露的个股权重偏小；穿透取其**主题成分股**用于质地判断。
  2. 数据集 `销售毛利率` 字段大面积缺失，已用 `主营业务利润率` 作代理。
  3. 部分公司经营现金流/净利润比率缺失或异常（如亏损公司），相应维度标 — 不计分。
  4. **Q8 安全边际与管理层诚信未量化**，评级只回答「这是不是好生意」，不回答「现在买贵不贵」。
  5. 凡数据不足或超出能力圈，一律 ⚪ 灰评，**绝不编造财务数字或硬凑结论**——这是产品的信任壁垒。
