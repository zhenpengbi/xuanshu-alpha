# GUARDRAILS.md — 玄枢Alpha 建设纪律

## 核心目标

玄枢的唯一目标：辅助毕老师做基金交易决策，带来可量化的理财收益。

## 三问准入检查（每个新需求必须通过）

1. 这个功能会改变操作建议吗？
2. 没有这个功能，会导致错误决策吗？
3. 优先级高于当前P0吗？

任一否定 → 排到P0之后。

## 当前P0清单

- [ ] 估值分位模块（解决"贵不贵"）
- [ ] 组合风险仪表盘（解决"风险多大"）
- [ ] 信号体系重构（趋势型改持有/定投，震荡型加估值确认）

## 禁止事项

- P0未完成时不接UI/美化/展示类需求
- 回测结论必须落地为产品动作
- 不得把"展示更多数据"等同于"决策能力提升"

## Claude Code 监督权

- 任务不符合准入条件时，有权拒绝执行
- 发现信号全部相同（如全"观望"）时，主动提醒
